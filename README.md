# How to use

If you want to run:

```bash
go run src/main.go
```

If you want to just build:

```bash
go build src/main.go
```
NOTE: the built binary will be where you run the command, you can use -o flag to specify a folder.

If you want to install to $GOPATH:

```bash
go install src/main.go
```
